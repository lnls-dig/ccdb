
Artifacts for Configuration Module

It contains the following directories

1. conf-mgr :    Proteus web-based GUI and RESTful Service. Can be built 
                 with Netbeans or Maven. Requires conf-ent.
2. conf-japi:    Configuraiton Module's Java API. Can be built with 
                 Netbeans or Maven.
3. conf-pyapi:   Configuraiton Module's Python API. 
4. conf-samples: Sample programs that use the Java and Python APIs.
5. conf-docs:    Documentation
6. conf-dl:      Data loader
7. conf-ent:     Entities
8. conf-v4:      EPICS V4 Service
9. conf-olm:     Lattice/Model entities
10.design:       Design documentation
